## Goal Keeper

A gamified, conversational AI coaching application that transforms vague
personal goals into structured, actionable daily tasks through a multi-agent
system.

### What it does

- Turns high-level goals ("get in shape", "learn guitar") into concrete daily
  tasks through a conversation with an AI coach.
- Uses a multi-agent system where agents propose changes and the user approves
  them — a human-in-the-loop design, so nothing happens without your sign-off.
- Privacy-first: data lives in the browser via localStorage, with optional
  Google sign-in.

### Tech

- **Backend:** Python (standard library only)
- **Frontend:** JavaScript
- **Storage:** browser localStorage

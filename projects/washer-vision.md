## Washer Vision

A web application that monitors washing machine status in shared laundry
facilities by capturing and analyzing digital clock displays from a camera
feed. Built for webJam 2024.

### What it does

- Points a camera at the machines' clock displays and reads them in real time.
- Detects whether a machine is running or free, so you don't have to walk down
  to the laundry room to check.
- Processes the camera feed with computer vision to interpret the displays.

### Tech

- **Frontend:** React
- **Backend:** Python / Flask
- **Vision:** OpenCV for real-time image processing
